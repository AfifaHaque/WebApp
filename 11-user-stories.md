# User Stories

## Account Management

* As a student, I want to create an account so that I can securely access my study planner.
* As a student, I want to log in to my account so that I can view and manage my academic information.

## Task Management

* As a student, I want to create tasks so that I can keep track of assignments and academic responsibilities.
* As a student, I want to edit existing tasks so that I can update task information when necessary.
* As a student, I want to delete tasks so that I can remove tasks that are no longer relevant.
* As a student, I want to mark tasks as completed so that I can track my progress.

## Deadline Tracking

* As a student, I want to set deadlines for tasks so that I can manage my time effectively.
* As a student, I want to view upcoming deadlines so that I do not miss important academic events.

## Calendar and Scheduling

* As a student, I want to view tasks on a calendar so that I can plan my schedule.
* As a student, I want to create study sessions so that I can allocate time for different courses.
* As a student, I want to use time-blocking features so that I can organize my study routine.

## Notifications

* As a student, I want to receive reminders for upcoming deadlines so that I can complete tasks on time.
* As a student, I want to receive notifications for scheduled study sessions so that I can follow my study plan.

## Progress Tracking

* As a student, I want to track completed tasks so that I can monitor my productivity.
* As a student, I want to view my study progress so that I can evaluate my academic performance.
