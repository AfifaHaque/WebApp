# Functional Requirements

## 1. Introduction

This document defines the functional requirements for the **Centralized Study Planner and Management System**, a web application designed to help university students manage academic responsibilities, track deadlines, organize study materials, and receive reminders for important events and examinations.

---

# 2. User Management

### FR-001: User Registration

The system shall allow students to create an account.

### FR-002: User Authentication

The system shall allow registered users to log in using valid credentials.

### FR-003: Password Recovery

The system shall provide a password recovery mechanism for users who forget their passwords.

### FR-004: User Logout

The system shall allow authenticated users to securely log out of their accounts.

---

# 3. Dashboard Management

### FR-006: Personalized Dashboard

The system shall provide a personalized dashboard displaying upcoming tasks, deadlines, exams, and events.

### FR-007: Academic Summary

The dashboard shall display a summary of pending tasks, completed tasks, and upcoming examinations.

### FR-008: Quick Access Navigation

The dashboard shall provide quick access to tasks, calendar events, study materials, and notifications.

---

# 4. Task Management

### FR-009: Create Tasks

The system shall allow users to create study-related tasks.

### FR-010: Edit Tasks

The system shall allow users to modify existing tasks.

### FR-011: Delete Tasks

The system shall allow users to delete tasks.

### FR-012: Task Categorization

The system shall allow users to categorize tasks by course, subject, or priority level.

### FR-013: Due Date Assignment

The system shall allow users to assign deadlines to tasks.

### FR-014: Task Completion Tracking

The system shall allow users to mark tasks as completed.

### FR-015: Task Status Monitoring

The system shall display the status of tasks as Pending, In Progress, or Completed.

---

# 5. Examination Management

### FR-016: Exam Schedule Creation

The system shall allow users to add examination schedules.

### FR-017: Exam Schedule Modification

The system shall allow users to update examination details.

### FR-018: Exam Schedule Deletion

The system shall allow users to remove examination schedules.

### FR-019: Exam Reminder Generation

The system shall automatically generate reminders for upcoming examinations.

### FR-020: Exam Overview

The system shall display all scheduled examinations in a dedicated view.

---

# 6. Study Material Management

### FR-021: Upload Study Materials

The system shall allow users to upload study materials including documents, PDFs, and lecture notes.

### FR-022: Organize Study Materials

The system shall allow users to organize uploaded materials by subject or course.

### FR-023: View Study Materials

The system shall allow users to access previously uploaded study materials.

### FR-024: Download Study Materials

The system shall allow users to download stored study materials.

### FR-025: Delete Study Materials

The system shall allow users to remove uploaded materials.

### FR-026: Search Study Materials

The system shall allow users to search for study materials using keywords.

---

# 7. Calendar Management

### FR-027: Calendar View

The system shall provide a calendar interface displaying academic activities.

### FR-028: Event Creation

The system shall allow users to create academic events.

### FR-029: Event Modification

The system shall allow users to edit existing events.

### FR-030: Event Deletion

The system shall allow users to remove events.

### FR-031: Event Visualization

The system shall display tasks, deadlines, examinations, and academic events within the calendar.

### FR-032: Date-Based Filtering

The system shall allow users to view events by day, week, or month.

---

# 8. Notification and Reminder Management

### FR-033: Deadline Notifications

The system shall notify users about approaching task deadlines.

### FR-034: Examination Notifications

The system shall notify users before scheduled examinations.

### FR-035: Event Notifications

The system shall notify users about upcoming academic events.

### FR-036: Notification Preferences

The system shall allow users to configure notification settings.

---

# 9. Search and Filtering

### FR-037: Global Search

The system shall provide a search feature for tasks, events, examinations, and study materials.

### FR-038: Subject-Based Filtering

The system shall allow users to filter information by subject or course.

### FR-039: Status-Based Filtering

The system shall allow users to filter tasks according to completion status.

---

# 10. Reporting and Analytics

### FR-040: Progress Tracking

The system shall provide users with a visual overview of completed and pending tasks.

### FR-041: Academic Activity Summary

The system shall generate summaries of upcoming deadlines, examinations, and events.

### FR-042: Productivity Insights

The system shall display statistics related to task completion and study activity.

---

# 11. Administrative Functions (Future Enhancement)

### FR-043: User Monitoring

The system shall allow administrators to monitor registered users.

### FR-044: Content Moderation

The system shall allow administrators to remove inappropriate content.

### FR-045: System Configuration

The system shall allow administrators to configure system-wide settings.
